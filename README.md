# Real Estate Market Trends Dashboard

## Project Overview

This Power BI dashboard analyzes real estate market trends using Excel data.

## Dashboard Features

- Total Properties
- Average Property Price
- Average Interest Rate
- Demand vs Supply Analysis
- Monthly Price Trend
- Property Type Distribution
- Properties by Region
- Area vs Price Analysis

## Tools Used

- Power BI
- Microsoft Excel
